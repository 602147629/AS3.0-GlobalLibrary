Classpublic class VizAlignTarget
InheritanceVizAlignTarget Object
Subclasses VizAlignGroup, VizAlignTextFieldTarget

The VizAlignTarget class represents a target for use in VizAlign. a VizAlignTarget handles many special aspects of aligning, such as retaining the original position of the target before alignment, the resulting position after alignment, and is capable of compensating for any wacky origins (or rotation point) of the given DisplayObject. the target must be a DisplayObject (that's the point of VizAlign!). in normal routine, you will not need to change any flags in this object. most will be taken care of in the VizAlign.align parameters. your biggest concern is if you wish to animate. to animate, the applyResults flag on VizAlign.align must be false. You then take the .end property of each target for the animation code. really simple.

Classpublic class VizAlign
InheritanceVizAlign Object

the VizAlign class is simple api for visually aligning DisplayObject. like the align panel in the flash ide, vizalign will align objects. however, with vizalign you are not limited to just the targets themselves or the stage. the object in which you align the targets to can be a Stage, DisplayObject, Rectangle, or the targets themselves. NOTE: almost all the methods other than align() are utilities that the align() uses. they are made public in case you want to utilize their functionality. more often than not you will not need to touch any of those methods. align()method
public static function align(targets:Array, vizAlignments:Array, ignoreOrigin:Boolean = false, applyResults:Boolean = false, pixelHinting:Boolean = false):Array

aligns the specified targets how the vizAlignments specify. this function is the main point of entry when using VizAlign. this is the function that will do _all_ the work for you. this is the only function in this library that will handle origins/rotation points completely for you. all you must do is specify the flag ignoreOrigin. Parameters

targets:Array — an array of DisplayObject/VizAlignTarget/VizAlignGroup/Array that indicates the targets to be aligned. no matter the state that they are sent in, they will all be converted to VizAlignTarget/VizAlignGroup accordingly.
 
vizAlignments:Array — an array of VizAlignment that specify how you want the targets to be aligned.
 
ignoreOrigin:Boolean (default = false) — whether you want VizAlign to actually move the targets or just calculate the end position
 
applyResults:Boolean (default = false) — whether you want VizAlign to ignore the origin offset.
 
pixelHinting:Boolean (default = false) — whether VizAlign should round the end result before finishing.

convertToVizAlignTargets()method 
public static function convertToVizAlignTargets(targets:Array):Array

will convert the objects in the argument to VizAlignTarget the elements of the paramater may be an Array, DisplayObject or VizAlignTarget. This is a recursive function. If you pass an array as an element to the array, this function assumes you are attempting to group objects and will recursively call this function. Thus, if you pass an array as an element to the paramater array, the same element requirements apply.

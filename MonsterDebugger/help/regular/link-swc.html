3. Linking the client SWC (to your Flash project)

Linking the Monster Debugger client SWC file to your project is easy and can be done in just a few steps depending on the IDE of choice. We've included examples for three popular IDE's. The starting point for each example is an empty project where you copy the client SWC file to a folder within your project called "libs".

Adobe Flash CS5

1/4 Image description: Go to the project properties tab and press the wrench icon on the right side of the script dropdown.

Adobe Flash Builder 4

1/1 Image description: If you already copied the MonsterDebugger.swc file to the libs directory within your project it should be up and running already. Good job Adobe!

FDT 4

1/4 Image description: Here is a basic empty project setup in FDT. You can see the Main document class and the MonsterDebugger.swc in the project explorer.

If you come across any difficulties linking the client SWC to your Flash project please let us know about it on our GetSatisfaction site so we can try to help you.
